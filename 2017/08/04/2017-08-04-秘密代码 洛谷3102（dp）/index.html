秘密代码

题目

对于一个字符串S只由大写字母组成 通过一系列操作对其进行加密操作为删除S的前面或者后面的若干个字符（但不删光整个S），并将剩下的部分连接到原字符串S的前面或者后面给出加密后的字符串 问共有几种加密方法？

输入输出格式

输入格式

一个字符串S （len<=100)

输出格式

几种加密方法

思路

由于长度较小可以使用暴力预处理字符的相同部分
f[i][j][k] 表示从i位置往后k个单位长度和从j位置往后k个单位长度是否相同

用dp思想进行区间动规 枚举删去后留下的长度进行更新
